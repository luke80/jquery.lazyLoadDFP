Jquery.lazyscript

Simple jquery plugin to load or transform html elements in lazy mode

Download .zip Download .tar.gz View on GitHub

This is a simple jquery plugin to execute any javascript code in lazy mode. You only have to provide a callback to execute for each element appearing in the viewport with a specific class.

It allows load images, videos, or any ajax request in a lazy way, making your web faster.

Demos:

  • Image lazy loader
  • Youtube videos lazy loader
